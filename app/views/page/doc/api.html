API文档

mat.env(env)

设置mat运行时的参数 

mat.env({
  port: 8080
})

port

type: Number

启动node服务运行的端口

mat.use(fn)

加载mat插件 

mat.use(opoa({
  root: './'
}))

fn

type: Generator Function

mat.task(name[, deps], fn)

定义任务 

mat.task('name', ['deps'], function() {
  //...
});

name

任务名称

deps

type: Array

当前任务依赖的任务，依赖任务会在当前任务被执行前执行

fn

type: Function
